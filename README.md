# HRMS Portal

MERN Stack Human Resource Management System developed during my internship to streamline HR operations and improve employee management processes.

## Project Overview
HRMS Portal is a web-based application designed to help organizations manage employees, attendance, leave requests, payroll, and user access through a secure role-based system.

## Features
- Employee Management
- Attendance Tracking
- Leave Management
- Payroll System
- Role-based Login
- Dashboard Analytics
- Secure Authentication using JWT
- Responsive User Interface
- Real-time Data Management

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js
- JavaScript
- HTML
- CSS
- Bootstrap

## Internship Experience
During my internship, I worked on both frontend and backend development of the HRMS Portal. I gained practical experience in full-stack web development, REST API integration, database management, authentication systems, and responsive UI design.

## Responsibilities
- Developed frontend pages using React.js
- Created backend APIs using Node.js and Express.js
- Managed employee and payroll data using MongoDB
- Implemented login authentication and role-based access
- Designed responsive dashboards and modules
- Tested and debugged application features
